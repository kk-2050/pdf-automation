What This Tool Does

This tool automatically searches for keywords inside PDF files, highlights them with a color, moves the processed files to a destination folder, and logs all results to a single Excel file.

Excel auto-reload on every Run — no need to restart the tool after changing settings.

Setup (First Time Only)
1

Install Python

Download Python 3.10 or later from python.org and install it.
During installation, check Add Python to PATH.

2

Extract the ZIP file to C:\PDFTool\

Right-click pdf_highlight_tool_v3.zip → "Extract All" → extract to C:\PDFTool\.

3

Install required libraries

Open Command Prompt and run:

cd C:\PDFTool

python -m pip install -r requirements.txt

💡 How to open Command Prompt
Press the Windows key → type "cmd" → press Enter.

Overwrite Option NEW
Checkbox stateBehaviorWhen to use
☐ OFF (default) Skip files that already exist in the Destination folder Normal processing (only process new PDFs)
☑ ON Delete existing files in Destination folder and reprocess When you changed page range, color, or keywords and want to reprocess
⚠ Warning when Overwrite is ON
Existing files in the Done folder will be permanently deleted before reprocessing.
Back up any important files before enabling this option.
✅ Example workflow
Change page range in Excel (e.g. 1–3 → 2–5) → Save Excel →
Check Overwrite in the tool → Click Run → Files are reprocessed with new settings.

Understanding Results

Log file: C:\PDFs\pdf_highlight_log.xlsx — all customers' results are recorded here.


StatusMeaningAction
OK_HITS_n Success. n keyword matches were found and highlighted. No action needed.
OK_HITS_0 Success. No keyword matches were found in the search range. Check keyword spelling or page range.
SKIPPED_EXISTS A file with the same name already exists in the Destination folder. Enable Overwrite and run again, or delete the file manually.
ERROR An error occurred during processing. Check if the PDF is open. See error message for details.
CANCELLED Processing was stopped by the user. Click Run again to process remaining files.

Frequently Asked Questions
Q

I changed the page range and want to reprocess

Edit the page range in Excel → Save → Enable Overwrite in the tool → Click Run.
No need to restart the tool.

Q

I want to search all pages

Leave the End_Page column blank. The tool will automatically search to the last page.

Q

Keywords are not found (OK_HITS_0)

• Check keyword spelling carefully.
• Verify that keywords appear within the Begin_Page to End_Page range.
• If the PDF was scanned from paper, it may not contain searchable text.

Q

What happens to the original PDF files?

Original files remain in the Input folder unchanged.
Only the highlighted copy is moved to the Done folder.

Q

Where is the log file?

All results are recorded in C:\PDFs\pdf_highlight_log.xlsx.
After processing, click the "Open Log" button to open it automatically.
