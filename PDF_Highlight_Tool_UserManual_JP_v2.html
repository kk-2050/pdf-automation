このツールでできること

このツールは、PDFファイルの中にあるキーワードを自動で探して、色をつけて（ハイライト）、別のフォルダに移動するツールです。

→

色をつける
→
📁 Done
移動する
→
📊 ログ
1つのExcelに記録
✅ v2.0の新機能
Single folder / All customers モード切り替えに対応。
Overwrite（上書き） オプションで再処理が簡単に。
共通ログファイル — 全顧客の処理結果を1つのExcelで管理。
RunするたびにExcelを自動再読み込み — ツールを再起動せず設定変更が反映される。
2
セットアップ（初回だけ）
1

Pythonをインストールする

python.org からPython 3.10以上をダウンロードしてインストール。
インストール時に Add Python to PATH にチェックを入れてください。

2

ZIPファイルを解凍して C:\PDFTool\ に置く

pdf_highlight_tool_v3.zip を右クリック →「すべて展開」→ C:\PDFTool\ に解凍します。

3

必要なライブラリをインストールする

コマンドプロンプトを開いて：

cd C:\PDFTool

python -m pip install -r requirements.txt

💡 コマンドプロンプトの開き方
Windowsキー → 「cmd」と入力 → Enter を押す。
3
フォルダの準備

以下のフォルダ構成を作ってください。

C:\PDFs\
  pdf_highlight_log.xlsx      ← 全顧客共通ログ（ここだけ確認すればOK）
  CustomerA\
    Input\                ← PDFをここに入れる
    Done\                 ← ハイライト済PDFが移動される
  CustomerB\
    Input\
    Done\
  CustomerC\
    Input\
    Done\
✅ ログファイルは1つだけ
C:\PDFs\pdf_highlight_log.xlsx に全顧客の処理結果がまとめて記録されます。
各顧客フォルダにLogフォルダは不要です。
⚠ 注意
PDFファイルは必ず Input フォルダに入れてください。
処理中はPDFファイルを開かないでください。
4
Excelマスターファイルの設定

highlight_master_v3.xlsx を開いて、Masterシートに入力します。

📊 highlight_master_v3.xlsx — Masterシート
CustomerCustomerFolderDestinationFolder KeywordsCaseSensitiveWholeWord ColorBegin_PageEnd_Page
CustomerAC:\PDFs\CustomerAC:\PDFs\CustomerA\Done Finalize,Opening,FactoryFALSEFALSE Blue13
CustomerBC:\PDFs\CustomerBC:\PDFs\CustomerB\Done Contract,Delivery,ApprovalFALSEFALSE Yellow13

列名内容例
Customer顧客名（ログに表示）CustomerA
CustomerFolder顧客フォルダのパス（InputフォルダはここのInput）C:\PDFs\CustomerA
DestinationFolderハイライト済PDFの移動先C:\PDFs\CustomerA\Done
Keywords検索キーワード（カンマ区切り、スペースなし）PPAP,IMDS,RoHS
CaseSensitive大文字小文字を区別するかFALSE
WholeWord完全一致のみ検索するかFALSE
Colorハイライトの色Blue / Yellow / Green
Begin_Page検索開始ページ（1から）1
End_Page検索終了ページ（空白=最後まで）3
✅ 設定変更はそのまま反映されます
Excelを保存してRunをクリックするだけで新しい設定が適用されます。
ツールを再起動する必要はありません。

ツールを起動する

run_highlight.bat をダブルクリックします。

2

Excelファイルを選ぶ（Step 01 Browse）

highlight_master_v3.xlsx を選択します。

3

Single folderを選択する

4

Inputフォルダを選ぶ（Step 02 Browse）

💡 Single folderの動作
選択したInputフォルダに一致する顧客のルールをExcelから自動で探して処理します。
例：C:\PDFs\CustomerA\Input を選択 → CustomerAのルールだけ処理

7
上書き処理（Overwrite） NEW
チェック状態動作使うタイミング
☐ OFF（デフォルト） Doneフォルダに同名ファイルがあればスキップ 通常の処理（新しいPDFだけ処理したい時）
☑ ON Doneフォルダの同名ファイルを削除して再処理 ページ範囲や色を変えて再処理したい時
⚠ Overwrite ONの注意
Doneフォルダの既存ファイルが削除されます。
必要なファイルは事前にバックアップしてください。
✅ 使い方の例
Excelでページ範囲を変更（例：1–3 → 2–5）→ Excelを保存 →
ツールでOverwriteをONにして Run → 新しいページ範囲で再処理される
8
処理結果の見方

ログファイル： C:\PDFs\pdf_highlight_log.xlsx — 全顧客の処理結果がここに記録されます。


ステータス意味対応
OK_HITS_n 処理成功。n件のキーワードが見つかった 問題なし
OK_HITS_0 処理成功。キーワードは見つからなかった キーワードのスペルやページ範囲を確認
SKIPPED_EXISTS 同名ファイルが移動先に存在するためスキップ Overwrite ONにして再実行、または手動で削除
ERROR エラーが発生した PDFが開いていないか確認。エラー内容を確認
CANCELLED Stopボタンで停止した 再度Runを押せば続きから処理できます

10
よくある質問
Q

ページ範囲を変えて再処理したい

Excelを開いてページ範囲を変更 → 保存 → ツールでOverwriteをONにして Run。
ツールの再起動は不要です。

Q

全ページを検索したい

End_Page 列を空白にしてください。自動で最後のページまで検索します。

Q

キーワードが見つからない（OK_HITS_0）

• キーワードのスペルを確認する
• Begin_Page / End_Page の範囲内にキーワードがあるか確認する
• PDFがスキャン画像の場合はテキストが含まれていない可能性があります

Q

原本PDFはどうなりますか？

原本はInputフォルダにそのまま残ります。
ハイライトされたコピーがDoneフォルダに移動します。

Q

ログファイルはどこにありますか？

C:\PDFs\pdf_highlight_log.xlsx に全顧客の処理結果が記録されています。
処理完了後に「Open Log」ボタンをクリックすると自動で開きます。

No PDF files foundInputフォルダにPDFがないPDFをInputフォルダに入れる
Permission deniedPDFが別のアプリで開かれているPDFを閉じてから再実行する
Missing required columnsExcel列名が正しくない列名がCustomer、CustomerFolder等と一致しているか確認
Input folder not foundInputフォルダのパスがExcelと違うCustomerFolderのパスとInputフォルダが存在するか確認

エラーメッセージをそのままコピーして担当者に送ってください。
